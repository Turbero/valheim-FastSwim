# Fast Swim

< Commissioned mod > 

Fast Swim will allow you to swim at faster speed when you press the Run button as same as you would do to run faster on the ground.

![](https://i.imgur.com/sw5MtGk.gif)

* When installed and by default, you will swim at double speed when pressing the Run button.
* Server-synced to apply to all players in servers.

## Usage instructions

Decide whether you want to override vanilla values or not and activate the fast swim. Configure the speed values for both in the .cfg file. You can modify:

- Swim depth
- Swim speed
- Swim turn speed
- Swim acceleration
